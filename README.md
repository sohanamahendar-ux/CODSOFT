This repository contains the projects completed as part of my **CodSoft Artificial Intelligence Internship (MAY BATCH B100)**.

During this internship, I worked on Artificial Intelligence projects using Python and applied various concepts such as rule-based systems, recommendation systems, and game-playing AI algorithms.

## Completed Tasks

### Task 1 – Rule-Based Chatbot

A simple chatbot developed using Python that responds to user inputs using predefined rules and conditions.

#### Features

* Greeting responses
* Date and time information
* Answers basic AI and Python-related questions
* Help menu support
* Exit functionality
* Default responses for unknown inputs

#### Concepts Used

* Python Functions
* Conditional Statements
* Loops
* User Input Handling
* Datetime Module
* Rule-Based AI

📁 Folder: `Task_Chatbot`


### Task 2 – Tic-Tac-Toe AI Using Minimax Algorithm

A terminal-based Tic-Tac-Toe game where the player competes against an AI opponent powered by the Minimax algorithm.

#### Features

* Human vs AI gameplay
* Intelligent move selection
* Win and draw detection
* Input validation
* Unbeatable AI strategy

#### Concepts Used

* Artificial Intelligence
* Minimax Algorithm
* Recursion
* Game Tree Search
* Functions and Loops

📁 Folder: `Task_Tic-Tac-Toe`


### Task 4 – Course Recommendation System

A content-based recommendation system that suggests beginner-friendly courses from different technology domains.

#### Features

* Multiple course categories
* Dynamic course ranking
* User rating system
* Real-time score updates
* AI-based category suggestions
* Session rating summary

#### Concepts Used

* Content-Based Filtering
* Recommendation Systems
* Bubble Sort Algorithm
* Functions and Lists
* Input Validation

📁 Folder: `Task_Course_Recommendation`

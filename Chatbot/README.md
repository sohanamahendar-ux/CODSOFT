# Chatbot - CodSoft Internship

This is my first task for the CodSoft Python Programming Internship. I made a simple chatbot using Python that can reply to basic messages, answer questions, and perform simple commands.

---

## About the Project

I built this chatbot using only Python and no external libraries except the datetime module, which comes built into Python. The chatbot works in the terminal and uses if-else statements to understand user input and provide appropriate responses.

This project helped me understand how rule-based chatbots work and gave me hands-on experience with Python concepts such as loops, conditions, user input handling, and working with dates and times.

---

## What the Chatbot can do

* Reply to hello and hi
* Respond when asked how it is doing
* Say bye and close the program
* Tell you today's date
* Tell you the current time
* Answer "what is python"
* Answer "what is ai"
* Answer "what is nlp"
* Answer "what is a chatbot"
* Answer "who created python"
* Show a help menu with all commands
* Give a default reply if it does not understand something

---

## How to Run

Make sure Python is installed on your system. You can download it from https://www.python.org

Then open Terminal or Command Prompt and run:

```bash
python chatbot.py
```

If that does not work, try:

```bash
python3 chatbot.py
```

You can also open the project in VS Code and run it from the integrated terminal.
